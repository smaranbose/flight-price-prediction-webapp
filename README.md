# Flight Price Prediction Web App

# Note

The model file is not included in this repository due to size constraints.
Please download it using the link below and place it in the project root directory before running the application.

# Description

A machine learning-powered web application that predicts airline ticket prices based on flight details such as airline, route, duration, class, stops, and days left before departure.

# Features

Predict flight prices in real-time using a trained ML model
Interactive web interface built with Flask
Handles categorical data using encoding pipeline
Displays predictions instantly based on user input
Feature importance visualization for model interpretability

# Machine Learning Model

Model: Random Forest Regressor
Framework: Scikit-learn

# Performance:

R² Score: ~0.98
MAE: ~1075
RMSE: ~2775
